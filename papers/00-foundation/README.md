# 00 Foundation (shared by both papers)

The work both papers stand on. Build it once, use it twice.

- `data/` is the dataset: the exposure layer, the search signal, the layoff
  weighting, and the frozen, documented snapshot. This is Paper 1's whole
  contribution and Paper 2's input.
- `reproducibility/` is the open pipeline and the replication package. It
  rebuilds every number from the frozen snapshot.
- `literature-map.md` is one literature landscape. It frames Paper 1's data
  contribution and Paper 2's predictive gap.

The program overview is in `../README.md`.
