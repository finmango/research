# Paper 1. Publication

A data or methods venue. The lane of our COVID open-data work.

## Where this goes

- A data journal that rewards a documented, reusable dataset (for example
  Scientific Data or Data in Brief).
- Or a software paper for the pipeline (for example JOSS), if the pipeline is the
  part that carries.
- Open by default. The dataset and the pipeline ship under MIT alongside the
  paper.

## Before we submit

- [ ] The dataset is frozen, versioned, and has a DOI (Zenodo).
- [ ] The data dictionary covers every field.
- [ ] The replication package rebuilds the dataset from scratch.
- [ ] Usage notes are honest: descriptive, no predictive or causal claim.
- [ ] The co-author has signed off.
- [ ] Funding and conflicts disclosed, per the brief.
