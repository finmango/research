# Paper 2. The leading indicator

The predictive study. Using Paper 1's frozen dataset, does the combined signal
lead realized occupational unemployment by one to three months in high-exposure
occupations. Pre-registered and falsifiable. Still measurement, not causation.

The steps, in order:

1. Claim. The one sentence, locked. See `claim.md`.
2. Literature gap. The shared landscape: `../00-foundation/literature-map.md`.
3. Pre-registration. Lock and post the design. See `preregistration/`. Post it
   before we pull any outcome data.
4. Data. Uses Paper 1's frozen dataset: `../00-foundation/data/`.
5. Analysis. The lead-lag test and the robustness checks. See `analysis-plan.md`.
6. Manuscript. The predictive draft. See `manuscript.md`. Don't draft it before
   the analysis is in.
7. Submit. A working paper first, then a journal. See `submission.md`.

Two rules we don't break: post the pre-registration before pulling the outcome
data, and don't draft the manuscript before the analysis is in.
